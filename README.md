## Project Description 

### Beer_Classification Kaggle: 
Kaggle competition using classical ML models like Decision Trees, Random Forests, and Logistic Regression along with libraries including scikit-learn to predict the quality of beer brands. The coding was completed in Python and the competition can be applied to industry classification. 

### Computer Vision Kaggle: 
Kaggle competition involving computer vision techniques like CNN, MLP, Logisitic Regression, KNN, SVMs, and libraries like scikit-learn and tensor-flow to predict a medical condition related to your retina. The coding was completed in Python and the competiton can be applied to any Computer Vision classification or similar type problems. 

### DRAG Graph Augmentation:
Applying different augmentation method including SMOTE, GraphSMOTE, GAN, GraphGAN, Time weighted edges, and Contrastive Learning to DRAG architecture to see if there is any improvement in detecting fradulent credit card transactions. Key libraries used include scikit-learn and pytorch. The coding was completed in Python and this project can be applied in any financial institution's fraud detection system. 

### Hockey Milestone 1:
Data Science project extracting play-by-play features for NHL hockey games across various seasons. The code for this was completed in Python. Moreover, the play-by-play project can be applied to any sport project which needs immediate statistical data. 

### Hockey Milestone 2:
Data Science project using Machine Learning methods to predict goals and results of hockey games using data from Milestone 1. The models used in this Milestone involved XGBoost, Logistic Regression, Calibration Curves, ROC curves, amongst other metrics along with libraries like scikit-learn. The code was completed with Python. 

### Hockey Milestone 3: 
Data Science project creating a website which predicts results of hockey games and types of goals scoed using information from Milestone 2. This project involves docker, flask apps, and other container codes which ensures that the models can properly be deployed to a website. The logic of deploying this model using flask is applicable to any model deployed to a website. The coding for this project was completed in Python.

Implemented MLPs, MobileNet CNNs, and U-Net segmentation architectures for image classification and semantic segmentation tasks, with professional applications in medical imaging, autonomous driving, and industrial visual inspection.
