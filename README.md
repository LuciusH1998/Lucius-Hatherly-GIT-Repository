Lucius Hatherly Personal GIT Repository 

Hello, my name is Lucius Hatherly and I am a professional working at the intersection of AI, MDM, and Data. Welcome to my own GITHub. This repository covers my coding projects both personal and those completed during my Master's in Machine Learning from the MILA program at the University of Montreal. 

Each branch covers its own coding project and contains its own readme file explaining the project and analyzed results. Below is a short description of each project found in the various branches. 

## Project Description 

### Beer_Classification Kaggle: 
Kaggle competition using classical ML models to predict the quality of beer brands. 

### Computer Vision Kaggle: 
Kaggle competition involving computer vision techniques to predict a medical condition related to your retina. 

### DRAG Graph Augmentation:
Applying different augmentation method to DRAG architecture to see if there is any improvement in detecting fradulent credit card transactions. 

### Hockey Milestone 1:
Data Science project extracting play-by-play features for NHL hockey games across various seasons. 

### Hockey Milestone 2:
Data Science project using MAchine Learning methods to predict goals and results of hockey games using data from Milestone 1. 

### Hockey Milestone 3: 
Data Science project creating a website which predicts results of hockey games and types of goals scoed using information from Milestone 2. 

### Representation Learning Assignment 1: 
Assignment which investigates different augmentation techniques in Deep Learning. 

### Representation Learning Assignment 2: 
Built GRU, attention, and Transformer-based sequential models for sentiment analysis and language modeling tasks.

### Representation Learning Assignment 3:
Implemented modern generative AI and alignment methods including DDPMs, Flow Matching, DPO, and reward modeling using PyTorch and GPT-2.
