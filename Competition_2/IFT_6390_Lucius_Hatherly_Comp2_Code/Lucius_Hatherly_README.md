## Overview/Introduction

This repository contains all code, preprocessing steps, and model implementations for Competition 2 (Retinal Image Classification).
The goal is to classify retinal fundus images into one of five diabetic retinopathy severity levels using machine-learning models.

Two Jupyter notebooks are included:

1. IFT_6390_Kaggle_Comp_2_MPP.ipynb

Minimal Preprocessing Pipeline (MPP)
Used for Milestone 1
Implements NumPy-only models:

KNN (custom)

Logistic Regression (custom)

Multiclass SVM (custom)

2. IFT_6390_Kaggle_Comp_2_APP.ipynb

Additional Preprocessing Pipeline (APP)
Used for both Milestone 1 (NumPy models) and Milestone 2 (scikit-learn + TensorFlow models).
Implements:

Numpy Logistic Regression 

Numpy KNN

Numpy SVM

Sklearn Logistic Regression (OVR)

Random Forest

MLP Classifier

CNN (TensorFlow/Keras)

The notebook clearly separates Milestone 1 and Milestone 2 sections.

Both notebooks automatically generate Kaggle-ready CSV submission files.

# Note on Requirements 
To properly execute the code, it is recommended to use Python 3.10.11, which is fully compatible with the TensorFlow version used in APP.ipynb.
Running the notebooks with Python 3.11 or later may cause compatibility issues with TensorFlow.
Jupyter Notebook or VS Code with a Python kernel is required to execute the .ipynb files.

## How to Run 

### Step 1: Data Preparation 

Before running either notebook

Download the official Kaggle data files including: 1. train_data.pkl, and 2. test_data.pkl

Create a data folder and place these two files inside the data folder. Competition_2/data/train_data.pkl
Competition_2/data/test_data.pkl

Both notebooks expect this exact directory structure.

### Step 2: Running the Notebooks

Option A: Run directly in VS Code (recommended)

No need to create an environment — simply ensure:

Python 3.10.11 is installed

VS Code Python extension is active

The kernel chosen matches your Python installation

Open up both IFT_6390_Kaggle_Comp_2_MPP.ipynb and IFT_6390_Kaggle_Comp_2_APP.ipynb in VS code and run the ipynb files. 

Option B: Create a dedicated virtual environment

python -m venv venv_tf
source venv_tf/bin/activate       # macOS/Linux
venv_tf\Scripts\activate          # Windows
pip install -r requirements.txt

This installs all required libraries:
NumPy, Pandas, Matplotlib, Seaborn, scikit-learn, TensorFlow, etc.

Open up both IFT_6390_Kaggle_Comp_2_MPP.ipynb and IFT_6390_Kaggle_Comp_2_APP.ipynb in VS code and run the ipynb files. 

### Step 3: Notebook Outputs

Running either .ipynb will:

Milestone 1 (MPP + APP)

Load and normalize the retinal images

Train:

KNN (NumPy)

Logistic Regression (NumPy)

SVM (NumPy)

Produce validation metrics and confusion matrices

Generate Milestone 1 Kaggle submission CSVs

Milestone 2 (APP only)

Apply brightness filtering & variance-based outlier removal

Standardize features

Train:

Logistic Regression (sklearn)

Random Forest

MLPClassifier

TensorFlow CNN

Perform GridSearchCV hyperparameter tuning

Generate:

Validation accuracy, error, classification reports

CNN training logs

Milestone 2 Kaggle submission CSVs
